python3 print("中文")异常

发表于 2018-09-18 | 分类于 python |

python print(“中文”)直接抛出异常处理

问题现象

　 在redhat7,Docker镜像中安装python3环境

直接异常，最开始没有时间去解决，我去掉了项目中的所有print语句．这是个偷懒的做法，哈哈．

问题原因分析

直接locale，查看编码，zh_CN.utf-8，没有问题

Google了一圈没有直接的解决办法，有个前辈说看下linux的编码转换库

果然，这台有问题的机器的编码转换很少，里面没有gb1830,gbk等
机智的同事跟踪了以下这个命令

发现调用的gblic

解决办法

再执行iconv -l 就发现多了好多编码了,python3 print(“中文”)也ok了．

# python3 # 中文 # 报错
